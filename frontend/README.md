# Frontend (Interactive Avatar)

该目录为 Interactive Avatar 前端应用（Vite + React 19）。

## 开发命令

```bash
npm install
npm run dev
```

默认开发端口：`5178`

## 构建命令

```bash
npm run build
npm run preview
```

## 环境变量

- `VITE_API_BASE`：后端地址，默认 `http://localhost:8788`

## 说明

- 前端主流程：`/create -> /rig-preview -> /scene-preview -> /interact`
- 详细项目说明、后端启动方式和完整接口请查看根目录 `README.md`
